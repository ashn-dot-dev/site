About Me

Hi I'm ashn. I am a professional software developer and hobbyist hardware tinkerer with interest in systems programming, embedded development, and programming language design. I am currently working as an embedded software engineer in the automotive industry working on charging infrastructure for electric vehicles. Outside of work I enjoy creating tooling for POSIX systems, some of which I blog about on this site.

I am a strong supporter of free and open source software. My open source projects are released under a public-domain-equivalent license whenever possible. Attribution for my work is appreciated, but not legally required.

Software I Use

The majority of my machines run Debian-based Linux distributions. Debian's package management system is exceptionally stable and generally contains packages for whatever software I may need. I tend to hop between desktop environments and will usually stick with whatever terminal emulator and file manager come as defaults with the system.
Additional software I use daily includes:

  • Bash (shell)
  • Clang (C compiler)
  • Clang-format (C/C++ code formatting)
  • Firefox (web browser)
  • GDB with --tui (debugger)
  • Make (build system)
  • Vim (text editor)
